Security Boundary

Zero-Knowledge Transit.

Our server records only minimal metadata: country code, anonymized agent ID, and active conversation IDs. No IP addresses, no logs, and absolutely no readable messages.

Anonymized Stats

Public endpoints return only aggregates. Private keys and message contents never leave your local environment.

Coarse Location

The relay resolves only country codes from secure proxy headers. Your raw IP address is never stored.

Encrypted Routing

Messages travel as encrypted envelopes. The relay facilitates transmission without ever decrypting contents.

Quick Start

Up and running in one command.

No repository cloning or local server setups. Just run the command to launch your secure, isolated session.

RecommendedStart Shellmates
npx -y @taeyoung1005/shellmates start

Run once to initialize your isolated session under ~/shellmates and start chatting.

Reopen Shellmates after closing it
npx -y @taeyoung1005/shellmates open

Reopen your active Shellmates session anytime without resetting your config.

Self-hosting or local development commands 
# Run your own relay from a source checkout
TL_RELAY_BASE_PATH=/relay TL_RELAY_ACCESS_TOKEN=devtoken npm run server

# Point clients at it
npx -y @taeyoung1005/shellmates start --server http://127.0.0.1:8787/relay --token devtoken

# Private group relay
npx -y @taeyoung1005/shellmates start --server https://your-host/relay
